Join the Blech community

Blech is an open source project. Anyone in the community can use and improve it. Here are few ways to learn about the community, and how to join.

Learn and Connect

Using or want to use Blech? Find out more here:

  • The Blech mailing list: Discuss with other Blech users!
  • Twitter @BlechLanguage: Follow us on Twitter to get the latest news!
  • Documentation on GitHub: Documentation is collected here!

Develop and Contribute

If you want to get more involved by contributing to Blech, join us here:

  • Blech compiler on Github: Compiler development takes place here!
  • Blech tools on Github: Tool development takes place here!
  • Collaborate on Slack: Collaborate on Blech development!

You can find out how to contribute to Blech development in our Contribution Guidelines.
